Callback functions

const running = (name) => console.log(`${name} is running `);
// this is a callback function. The function is making a call back to a different function. 
doActionWithSomeone(running, "Anders");
oneliner callback 
doActionWithSomeoneOneLiner((name) => (console.log(`${name} is doing`), 'Frederik'));
doActionWithSomeone(name => (console.log(`${name} anything`)), "stuff");

Hoisting

is a part of node/javascript that pulls the necessary function called upon to the top
an example that we used is 
console.log(add(2, 8));
function add(x, y) {
    return x + y;
}
the function add, will be run before the console.log, and thus it will give the result of 10 and work correctly.

Package.json

java - pom.xml                      node/javascript - package.json
package.json is a file the consists of the the dependencies that the program is supposed to install when "npm i" is ran.
on a side note a json file must be valid json by using curly brackets {}
It also consists of eventual scripts like "npm run start-dev"
Use `npm install "pkg"` afterwards to install a package and
save it as a dependency in the package.json file.
